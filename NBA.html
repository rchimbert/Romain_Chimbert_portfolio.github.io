Project overview

This end-to-end project consists in gathering some statistics by nba franchises such as the all-time top 5 scorer, the top 5 rebounders, the number of championships won by the franchise, or their three point shooting throughout the ages on an interactive dashboard. One of the main difficulty is that some franchises have change names several times during their history, so we have to group those statistics under the same franchise code.

Data gathering, cleaning and exporting to csv file with python

First, I gathered some data of players attribute and player statistics by seasons from 1950 to 2017 thanks to a Kaggle dataset available online (The dataset, I used the season_stat.csv and player_data.csv). Then, I gathered some more datas from a website gathering every NBA champions. I then cleaned and processed the data in order to make further SQL request then. This step is detailed on this code available on my Github: The code.

Creation of SQL table and filling them with the data rows

I first created the structures of the table used for the SQL request and made few modifications of some tables element directly on SQL. Then, I created another table to gather every NBA franchises former name(s) in order to count the number of championship won by the franchise and some all-time statistics such as the best scorers. This step is detailed on this code available on my Github: The code.

SQL queries in order to fill the Tableau Dashboard with statistics

To get some more insight about this seasons statistics dataset by player, I created several SQL queries on postgreSQL to be shown then in a Tableau Dashboard. This step is detailed on this code available on my Github: The code.

Tableau dashboard visual

Finally, I was able to create a visual available on my Tableau Public profile. On this dashboard, we can select an NBA franchise at the top left corner, and it displays the top 5 all-time scorer, rebounder, passer, blocker by franchise and the number of championships won. Every player attributes (height and weight) are visible by dragging the mouse on their name. This interactive visual is available here on my Tableau Public profile (note: if you are on laptop think about ticking the option of displaying the dashboard as on a desktop at the bottom): The dashboard.
